<h2 align="center">Sobre 📖</h2>

---
> - ### Projeto Full Stack.
> - ### O objetivo é criar uma plataforma de anúncios de produtos, inspirada na Olx.
---
<h2 align="center">Tecnologias e Bibliotecas Utilizadas 📱</h2>

> ## -> BackEnd
> - NodeJs
> - TypeScript
> - Banco de Dados -> MySQL
> - Express
> - Dotenv
> - Sequelize
> - Validator
> - Sharp
> - Passport-jwt
> - Passport
> - mysql2
> - Multer
> - Jsonwebtoken
> - Cors
> - Bcrypt
> - Nodemon
> ## -> FrontEnd
> - JavaScript
> - React
> - Styled-Components
> - React-Text-Mask
> - Text-Mask-Addons
> - React-Slideshow-Image
> - React-Router-Dom
> - Pagix
> - Js-Cookie
> - Axios 
> - QS
---
<h2 align="center">Funcionalidades 🛠️</h2>

> ## BackEnd
> - Rotas Privadas
> - Upload de imagens
> - RESTful API
> - Padrão MVC
> ## FrontEnd
> - Login
> - Cadastro de Usuário
> - Postar novo anúncio
> - Editar anúncios
> - Ativar/desativar anúncios
> - Editar dados do usuário
> - Deletar imagens dos anúncios

---
<h2 align="center">Funcionalidades a Serem Implementadas🛠️</h2>
